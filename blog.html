Var and let can be declared without initialization. Constant cannot be declared without initialization. var declarations are globally scoped or function/locally scoped. Let is only available for use within that block. Like let declarations, const declarations can only be accessed within the block they were declared.

In arrow function this keyword means the global execution window. But in the regular function this keyword means the current object.

Map iterates over all the values of a array and after performing some operation and return the result array. On the other hand forEach does the same but it does not return anything.filter() returns an array containing the element that satisfies the condition, but find() returns the element itself that satisfies the condition.In filter(), whole array is iterated despite the fact that the element being searched for is present at the beginning. But in find(), as soon as the element that satisfies the condition is found, it gets returned.
